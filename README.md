Pregnancy Risk Prediction

This repository contains a machine learning project designed to predict pregnancy risk percentages based on patient data. It features a data training pipeline, a user-friendly graphical web application, and a command-line interface tool.

🌟 Features

Web Interface: A user-friendly web app built with Streamlit for intuitive data entry and result visualization.


Command-Line Interface: A lightweight, terminal-based Python script for predicting risk via standard input prompts.


Automated Calculations: Automatically calculates BMI from height (feet/inches) and weight (kg), and converts blood sugar from mmol/L to glucose in mg/dL.


Robust Model Selection**: Includes a training script that compares Random Forest, Decision Tree, and Logistic Regression models. It automatically evaluates them using a custom overall score (combining accuracy, precision, recall, and F1) and saves the best-performing pipeline.



📊 Dataset

The model is trained on dataset_1.csv. The original data is sourced from the [Maternal Health and High-Risk Pregnancy Dataset on Kaggle](https://www.kaggle.com/datasets/vmohammedraiyyan/maternal-health-and-high-risk-pregnancy-dataset).

📁 Project Structure

`app.py`: The Streamlit web application.


`best_model_dataset_1.pkl`: The serialized, pre-trained model pipeline generated by the training script.


`requirements.txt`: The list of required Python dependencies.


*Additional included Python scripts allow for training the model and evaluating data via a command-line interface.



🚀 How to Run

1. Install dependencies**
Make sure you have Python installed, then install the required libraries:


pip install -r requirements.txt


2. Use the cd (change directory) command to navigate to the folder where you saved your project files (specifically app.py and requirements.txt).

Example: cd Documents/pregnancy-risk-project


3. Run the Streamlit Web App**
Start the local Streamlit server:


streamlit run app.py


Once running, simply fill in the medical and personal information form in your browser and click "Predict Risk" to see the prediction.
